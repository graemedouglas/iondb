# Phase 2: WAL Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the write-ahead log (`iondb-wal`) with two storage layouts (flat, page-segmented), four sync policies, three truncation modes, and iterator-based crash recovery.

**Architecture:** WAL is generic over `IoBackend`, depends only on `iondb-core`. Record format is a 29-byte header (magic, CRC, LSN, TxnId, type, key_len, val_len) + variable payload. Layout and truncation mode are user-configurable. Recovery uses a `next_record()` reader pattern (not `Iterator` trait) for zero-alloc `no_std` compatibility.

**Tech Stack:** Rust, `no_std`, `iondb-core` (CRC, endian, page, types, error), proptest, cargo-fuzz, `FailpointIoBackend` from `iondb-io`

**Spec:** `docs/superpowers/specs/2026-04-05-phase2-wal-design.md`

//! WAL configuration types: sync policy, storage layout, truncation mode.

use iondb_core::error::{Error, Result};

/// Sync policy for WAL writes.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum SyncPolicy {
    /// Sync after every record append.
    EveryRecord,
    /// Sync after Commit/Rollback records only.
    EveryTransaction,
    /// Sync every N record appends.
    Periodic(u32),
    /// Caller controls sync manually.
    None,
}

/// WAL storage layout.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum WalLayout {
    /// Records appended back-to-back as a raw byte stream.
    Flat,
    /// Records packed into fixed-size pages with headers and checksums.
    PageSegmented {
        /// Page size in bytes (must be >= `PAGE_OVERHEAD + RECORD_HEADER_SIZE`).
        page_size: usize,
    },
}

/// Truncation mode for WAL space reclamation.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum TruncationMode {
    /// Logical only: track checkpoint LSN, no space reclaimed.
    Logical,
    /// Logical + physical compaction. Only available with `std`.
    #[cfg(feature = "std")]
    Physical,
    /// Fixed-size circular buffer with wrap-around.
    Circular {
        /// Total capacity in bytes (includes 32-byte header).
        capacity: usize,
    },
}

/// WAL configuration, validated at construction time.
#[derive(Debug, Clone)]
pub struct WalConfig {
    /// Storage layout.
    pub layout: WalLayout,
    /// Sync policy.
    pub sync_policy: SyncPolicy,
    /// Truncation mode.
    pub truncation: TruncationMode,
}

/// Minimum capacity for circular mode: 32-byte header + room for at least
/// one minimum record (29-byte header).
const CIRCULAR_MIN_CAPACITY: usize = 32 + 29;

impl WalConfig {
    /// Validate this configuration, returning an error for invalid combinations.
    ///
    /// Invalid combinations:
    /// - `Circular` + `PageSegmented` (not supported)
    /// - `PageSegmented` with page_size too small for one record header
    /// - `Circular` with capacity too small for header + one record
    /// - `Periodic(0)` (would sync every 0 records — meaningless)
    ///
    /// # Errors
    ///
    /// Returns [`Error::WalError`] if the config is invalid.
    pub fn validate(&self) -> Result<()> {
        // Circular + PageSegmented is not supported
        if matches!(self.truncation, TruncationMode::Circular { .. })
            && matches!(self.layout, WalLayout::PageSegmented { .. })
        {
            return Err(Error::WalError);
        }

        // PageSegmented page_size must fit at least the overhead + record header
        if let WalLayout::PageSegmented { page_size } = self.layout {
            let min_page = iondb_core::page::PAGE_OVERHEAD + crate::record::RECORD_HEADER_SIZE;
            if page_size < min_page {
                return Err(Error::WalError);
            }
        }

        // Circular capacity must fit header + at least one record
        if let TruncationMode::Circular { capacity } = self.truncation {
            if capacity < CIRCULAR_MIN_CAPACITY {
                return Err(Error::WalError);
            }
        }

        // Periodic(0) is meaningless
        if matches!(self.sync_policy, SyncPolicy::Periodic(0)) {
            return Err(Error::WalError);
        }

        Ok(())
    }
}

```rust
//! WAL record format: serialization, deserialization, CRC integrity.
//!
//! ## On-disk format (29-byte fixed header + variable payload)
//!
//! ```text
//! Offset  Size  Field
//! ------  ----  -----
//!  0      2     magic (0x57 0x4C, "WL")
//!  2      4     crc32 (IEEE, over bytes 6..end)
//!  6      8     lsn (little-endian u64)
//! 14      8     txn_id (little-endian u64)
//! 22      1     record_type (u8)
//! 23      2     key_len (little-endian u16)
//! 25      4     val_len (little-endian u32)
//! 29      var   key (key_len bytes)
//! 29+K    var   value (val_len bytes)
//! ```

use iondb_core::crc;
use iondb_core::endian;
use iondb_core::error::{Error, Result};
use iondb_core::types::{Lsn, TxnId};

/// Magic bytes at the start of every WAL record: "WL" (0x57, 0x4C).
pub const MAGIC: [u8; 2] = [0x57, 0x4C];

/// Size of the fixed record header in bytes.
pub const RECORD_HEADER_SIZE: usize = 29;

/// Offset of the CRC field within the record.
const CRC_OFFSET: usize = 2;

/// Offset where the CRC-covered region begins.
const CRC_DATA_START: usize = 6;

/// WAL record types.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
#[repr(u8)]
pub enum RecordType {
    /// Transaction begin marker.
    Begin = 0,
    /// Key-value write.
    Put = 1,
    /// Key deletion.
    Delete = 2,
    /// Transaction committed.
    Commit = 3,
    /// Transaction rolled back.
    Rollback = 4,
    /// Checkpoint marker.
    Checkpoint = 5,
}

impl RecordType {
    /// Convert a raw byte to a [`RecordType`].
    ///
    /// # Errors
    ///
    /// Returns [`Error::WalError`] if the byte is not a valid record type.
    pub fn from_byte(b: u8) -> Result<Self> {
        match b {
            0 => Ok(Self::Begin),
            1 => Ok(Self::Put),
            2 => Ok(Self::Delete),
            3 => Ok(Self::Commit),
            4 => Ok(Self::Rollback),
            5 => Ok(Self::Checkpoint),
            _ => Err(Error::WalError),
        }
    }

    /// Convert to the raw byte representation.
    #[must_use]
    pub fn as_byte(self) -> u8 {
        self as u8
    }
}

/// A WAL record with key/value data borrowed from a buffer.
#[derive(Debug, PartialEq, Eq)]
pub struct WalRecord<'a> {
    /// Log sequence number.
    pub lsn: Lsn,
    /// Transaction identifier.
    pub txn_id: TxnId,
    /// Record type.
    pub record_type: RecordType,
    /// Key data (may be empty for Begin/Commit/Rollback/Checkpoint).
    pub key: &'a [u8],
    /// Value data (may be empty for Delete/Begin/Commit/Rollback/Checkpoint).
    pub value: &'a [u8],
}

/// Compute the total serialized size of a record.
#[must_use]
pub fn record_size(key_len: usize, val_len: usize) -> usize {
    RECORD_HEADER_SIZE + key_len + val_len
}

/// Serialize a WAL record into `buf`.
///
/// Returns the number of bytes written (`RECORD_HEADER_SIZE + key.len() + value.len()`).
///
/// # Errors
///
/// Returns [`Error::WalError`] if `buf` is too small.
pub fn serialize_into(
    buf: &mut [u8],
    lsn: Lsn,
    txn_id: TxnId,
    record_type: RecordType,
    key: &[u8],
    value: &[u8],
) -> Result<usize> {
    let total = record_size(key.len(), value.len());
    if buf.len() < total {
        return Err(Error::WalError);
    }

    // Magic
    buf[0] = MAGIC[0];
    buf[1] = MAGIC[1];
    // Skip CRC (fill after writing data)

    // LSN
    endian::write_u64_le(&mut buf[6..], lsn)?;
    // TxnId
    endian::write_u64_le(&mut buf[14..], txn_id)?;
    // RecordType
    endian::write_u8(&mut buf[22..], record_type.as_byte())?;
    // KeyLen
    let key_len: u16 = key.len().try_into().map_err(|_| Error::WalError)?;
    endian::write_u16_le(&mut buf[23..], key_len)?;
    // ValLen
    let val_len: u32 = value.len().try_into().map_err(|_| Error::WalError)?;
    endian::write_u32_le(&mut buf[25..], val_len)?;
    // Key
    buf[RECORD_HEADER_SIZE..RECORD_HEADER_SIZE + key.len()].copy_from_slice(key);
    // Value
    let val_start = RECORD_HEADER_SIZE + key.len();
    buf[val_start..val_start + value.len()].copy_from_slice(value);

    // CRC over bytes 6..total
    let checksum = crc::crc32(&buf[CRC_DATA_START..total]);
    endian::write_u32_le(&mut buf[CRC_OFFSET..], checksum)?;

    Ok(total)
}

/// Deserialize a WAL record from `buf`.
///
/// Validates magic bytes and CRC. On success, returns a `WalRecord` that
/// borrows key/value slices from `buf`.
///
/// # Errors
///
/// - [`Error::WalError`] if magic bytes don't match or header is incomplete.
/// - [`Error::Corruption`] if CRC doesn't match.
pub fn deserialize_from(buf: &[u8]) -> Result<WalRecord<'_>> {
    if buf.len() < RECORD_HEADER_SIZE {
        return Err(Error::WalError);
    }

    // Check magic
    if buf[0] != MAGIC[0] || buf[1] != MAGIC[1] {
        return Err(Error::WalError);
    }

    // Read lengths to determine total size
    let key_len = endian::read_u16_le(&buf[23..])? as usize;
    let val_len = endian::read_u32_le(&buf[25..])? as usize;
    let total = record_size(key_len, val_len);

    if buf.len() < total {
        return Err(Error::WalError);
    }

    // Verify CRC
    let stored_crc = endian::read_u32_le(&buf[CRC_OFFSET..])?;
    let computed_crc = crc::crc32(&buf[CRC_DATA_START..total]);
    if stored_crc != computed_crc {
        return Err(Error::Corruption);
    }

    // Read fields
    let lsn = endian::read_u64_le(&buf[6..])?;
    let txn_id = endian::read_u64_le(&buf[14..])?;
    let record_type = RecordType::from_byte(endian::read_u8(&buf[22..])?)?;
    let key = &buf[RECORD_HEADER_SIZE..RECORD_HEADER_SIZE + key_len];
    let value = &buf[RECORD_HEADER_SIZE + key_len..total];

    Ok(WalRecord {
        lsn,
        txn_id,
        record_type,
        key,
        value,
    })
}

/// Read only the fixed header fields without CRC validation.
///
/// Used by `CommittedRecoveryReader` pass 1 to scan for Commit/Rollback
/// records without reading key/value data. Reads only the 29-byte header.
///
/// Returns `(lsn, txn_id, record_type, key_len, val_len)`.
///
/// # Errors
///
/// Returns [`Error::WalError`] if header is incomplete or magic is wrong.
pub fn read_header(buf: &[u8]) -> Result<(Lsn, TxnId, RecordType, u16, u32)> {
    if buf.len() < RECORD_HEADER_SIZE {
        return Err(Error::WalError);
    }
    if buf[0] != MAGIC[0] || buf[1] != MAGIC[1] {
        return Err(Error::WalError);
    }
    let lsn = endian::read_u64_le(&buf[6..])?;
    let txn_id = endian::read_u64_le(&buf[14..])?;
    let record_type = RecordType::from_byte(endian::read_u8(&buf[22..])?)?;
    let key_len = endian::read_u16_le(&buf[23..])?;
    let val_len = endian::read_u32_le(&buf[25..])?;
    Ok((lsn, txn_id, record_type, key_len, val_len))
}

//! Flat (sequential) WAL layout: write, read, magic-byte scan, circular header.
//!
//! Records are appended back-to-back to the `IoBackend`. Recovery after
//! corruption uses magic-byte scanning to find the next valid record.

use iondb_core::crc;
use iondb_core::endian;
use iondb_core::error::{Error, Result};
use iondb_core::traits::io_backend::IoBackend;
use iondb_core::types::Lsn;

use crate::record::{self, MAGIC, RECORD_HEADER_SIZE};

/// Size of the circular buffer header block in bytes.
pub const CIRCULAR_HEADER_SIZE: usize = 32;

/// Magic bytes for the circular header: "WLCR" (0x57, 0x4C, 0x43, 0x52).
pub const CIRCULAR_MAGIC: [u8; 4] = [0x57, 0x4C, 0x43, 0x52];

/// Circular buffer header, stored at offset 0.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub struct CircularHeader {
    /// Next write position.
    pub head_offset: u64,
    /// Oldest live record position.
    pub tail_offset: u64,
    /// Checkpoint LSN.
    pub checkpoint_lsn: Lsn,
}

impl CircularHeader {
    /// Encode the header into `buf` (must be >= `CIRCULAR_HEADER_SIZE`).
    pub fn encode(&self, buf: &mut [u8]) -> Result<()> {
        if buf.len() < CIRCULAR_HEADER_SIZE {
            return Err(Error::WalError);
        }
        buf[0..4].copy_from_slice(&CIRCULAR_MAGIC);
        // Skip CRC at 4..8, fill after
        endian::write_u64_le(&mut buf[8..], self.head_offset)?;
        endian::write_u64_le(&mut buf[16..], self.tail_offset)?;
        endian::write_u64_le(&mut buf[24..], self.checkpoint_lsn)?;
        // CRC over bytes 8..32
        let checksum = crc::crc32(&buf[8..CIRCULAR_HEADER_SIZE]);
        endian::write_u32_le(&mut buf[4..], checksum)?;
        Ok(())
    }

    /// Decode the header from `buf`.
    pub fn decode(buf: &[u8]) -> Result<Self> {
        if buf.len() < CIRCULAR_HEADER_SIZE {
            return Err(Error::WalError);
        }
        if buf[0..4] != CIRCULAR_MAGIC {
            return Err(Error::WalError);
        }
        let stored_crc = endian::read_u32_le(&buf[4..])?;
        let computed_crc = crc::crc32(&buf[8..CIRCULAR_HEADER_SIZE]);
        if stored_crc != computed_crc {
            return Err(Error::Corruption);
        }
        let head_offset = endian::read_u64_le(&buf[8..])?;
        let tail_offset = endian::read_u64_le(&buf[16..])?;
        let checkpoint_lsn = endian::read_u64_le(&buf[24..])?;
        Ok(Self {
            head_offset,
            tail_offset,
            checkpoint_lsn,
        })
    }
}

/// Write a serialized record to the backend at `offset`.
///
/// Returns the offset immediately after the written record.
///
/// # Errors
///
/// Returns `Error::Io` if the write fails.
pub fn write_record(backend: &mut impl IoBackend, offset: u64, data: &[u8]) -> Result<u64> {
    let written = backend.write(offset, data)?;
    if written != data.len() {
        return Err(Error::Io);
    }
    Ok(offset + data.len() as u64)
}

/// Read a record from the backend at `offset` into `buf`.
///
/// Returns `Some((record, next_offset))` on success, `None` if end-of-log
/// (no valid magic at offset), or `Err` on I/O or corruption.
///
/// On CRC mismatch, returns `Err(Error::Corruption)` — the caller
/// should use `scan_for_magic` to find the next record.
pub fn read_record<'buf>(
    backend: &impl IoBackend,
    offset: u64,
    end: u64,
    buf: &'buf mut [u8],
) -> Result<Option<(record::WalRecord<'buf>, u64)>> {
    // Need at least a header
    if offset + RECORD_HEADER_SIZE as u64 > end {
        return Ok(None);
    }

    // Read the header first to learn the record size
    let header_read = backend.read(offset, &mut buf[..RECORD_HEADER_SIZE])?;
    if header_read < RECORD_HEADER_SIZE {
        return Ok(None);
    }

    // Check magic — if not present, this is end-of-log (not corruption)
    if buf[0] != MAGIC[0] || buf[1] != MAGIC[1] {
        return Ok(None);
    }

    // Read key_len and val_len to determine full size
    let key_len = endian::read_u16_le(&buf[23..])? as usize;
    let val_len = endian::read_u32_le(&buf[25..])? as usize;
    let total = record::record_size(key_len, val_len);

    if offset + total as u64 > end {
        return Ok(None); // Truncated record — treat as end-of-log
    }

    if buf.len() < total {
        return Err(Error::WalError); // Caller's buffer too small
    }

    // Read the remaining payload (key + value) if any
    if total > RECORD_HEADER_SIZE {
        let payload_read = backend.read(
            offset + RECORD_HEADER_SIZE as u64,
            &mut buf[RECORD_HEADER_SIZE..total],
        )?;
        if payload_read < total - RECORD_HEADER_SIZE {
            return Ok(None); // Truncated
        }
    }

    // Deserialize (validates CRC)
    let rec = record::deserialize_from(&buf[..total])?;
    Ok(Some((rec, offset + total as u64)))
}

/// Scan forward from `start` looking for magic bytes `0x57 0x4C`.
///
/// Returns the offset of the first magic byte match, or `None` if
/// the end is reached. Note: callers must still validate CRC after
/// finding a magic match (the bytes could appear in payload data).
pub fn scan_for_magic(
    backend: &impl IoBackend,
    start: u64,
    end: u64,
) -> Result<Option<u64>> {
    let mut offset = start;
    let mut byte = [0u8; 1];
    while offset + 1 < end {
        let n = backend.read(offset, &mut byte)?;
        if n == 0 {
            return Ok(None);
        }
        if byte[0] == MAGIC[0] {
            // Check second byte
            let n2 = backend.read(offset + 1, &mut byte)?;
            if n2 > 0 && byte[0] == MAGIC[1] {
                return Ok(Some(offset));
            }
        }
        offset += 1;
    }
    Ok(None)
}

/// Write the circular header to the backend at offset 0.
pub fn write_circular_header(
    backend: &mut impl IoBackend,
    header: &CircularHeader,
) -> Result<()> {
    let mut buf = [0u8; CIRCULAR_HEADER_SIZE];
    header.encode(&mut buf)?;
    let written = backend.write(0, &buf)?;
    if written != CIRCULAR_HEADER_SIZE {
        return Err(Error::Io);
    }
    Ok(())
}

/// Read the circular header from the backend at offset 0.
pub fn read_circular_header(backend: &impl IoBackend) -> Result<CircularHeader> {
    let mut buf = [0u8; CIRCULAR_HEADER_SIZE];
    let n = backend.read(0, &mut buf)?;
    if n < CIRCULAR_HEADER_SIZE {
        return Err(Error::WalError);
    }
    CircularHeader::decode(&buf)
}

```rust
//! Page-segmented WAL layout: page packing, page-boundary recovery.
//!
//! Records are packed into fixed-size `WalSegment` pages. Each page has
//! a 16-byte header and a 4-byte trailing checksum (see `iondb_core::page`).
//! Records do NOT span page boundaries — if a record doesn't fit, the
//! current page is finalized and a new page is started.

use iondb_core::error::{Error, Result};
use iondb_core::page::{self, PageHeader, PageType, PAGE_CHECKSUM_SIZE, PAGE_HEADER_SIZE, PAGE_OVERHEAD};
use iondb_core::traits::io_backend::IoBackend;
use iondb_core::types::PageId;

use crate::record::{self, RECORD_HEADER_SIZE};

/// State for writing records into a page-segmented layout.
#[derive(Debug)]
pub struct PagedWriter {
    page_size: usize,
    /// Offset of the current page in the backend.
    page_offset: u64,
    /// Current write position within the page (relative to page start).
    pos_in_page: usize,
    /// Next page_id to assign.
    next_page_id: PageId,
}

impl PagedWriter {
    /// Create a new writer starting at `start_offset`.
    pub fn new(page_size: usize, start_offset: u64) -> Self {
        Self {
            page_size,
            page_offset: start_offset,
            pos_in_page: PAGE_HEADER_SIZE, // skip header, write records after it
            next_page_id: 0,
        }
    }

    /// Resume a writer at a known position (used by `Wal::open()`).
    pub fn resume(page_size: usize, page_offset: u64, pos_in_page: usize, next_page_id: PageId) -> Self {
        Self {
            page_size,
            page_offset,
            pos_in_page,
            next_page_id,
        }
    }

    /// Usable payload space per page (excluding header and checksum).
    fn usable(&self) -> usize {
        self.page_size - PAGE_OVERHEAD
    }

    /// Remaining space in the current page for records.
    fn remaining_in_page(&self) -> usize {
        // Space between current position and where the checksum will go
        let checksum_start = self.page_size - PAGE_CHECKSUM_SIZE;
        if self.pos_in_page >= checksum_start {
            0
        } else {
            checksum_start - self.pos_in_page
        }
    }

    /// Write a serialized record into pages.
    ///
    /// If the record doesn't fit in the current page, finalizes the current
    /// page (writes checksum) and starts a new one.
    ///
    /// Returns the byte offset in the backend where the record was written.
    pub fn write_record(
        &mut self,
        backend: &mut impl IoBackend,
        record_data: &[u8],
    ) -> Result<u64> {
        let record_len = record_data.len();

        // Record must fit in a single page's usable space
        if record_len > self.usable() {
            return Err(Error::WalError);
        }

        // If it doesn't fit in current page, finalize and start new page
        if record_len > self.remaining_in_page() {
            self.finalize_page(backend)?;
            self.start_new_page(backend)?;
        }

        // If this is the very first write (no page started yet), start a page
        if self.pos_in_page == PAGE_HEADER_SIZE && self.page_offset == 0 && self.next_page_id == 0 {
            self.start_new_page(backend)?;
        }

        let record_offset = self.page_offset + self.pos_in_page as u64;
        let written = backend.write(record_offset, record_data)?;
        if written != record_len {
            return Err(Error::Io);
        }
        self.pos_in_page += record_len;
        Ok(record_offset)
    }

    /// Finalize the current page: write page checksum.
    pub fn finalize_page(&mut self, backend: &mut impl IoBackend) -> Result<()> {
        // Read the entire page, compute checksum, write it
        // We need a buffer for the page. Since we can't allocate,
        // read/write the checksum directly.
        let data_len = self.page_size - PAGE_CHECKSUM_SIZE;
        // Read page data to compute checksum
        // For efficiency, compute CRC incrementally by reading in chunks
        let mut crc_state = 0xFFFF_FFFFu32;
        let mut chunk = [0u8; 64];
        let mut read_off = self.page_offset;
        let end_off = self.page_offset + data_len as u64;
        while read_off < end_off {
            let to_read = chunk.len().min((end_off - read_off) as usize);
            let n = backend.read(read_off, &mut chunk[..to_read])?;
            if n == 0 {
                break;
            }
            crc_state = iondb_core::crc::crc32_update(crc_state, &chunk[..n]);
            read_off += n as u64;
        }
        let checksum = crc_state ^ 0xFFFF_FFFF;
        let mut crc_buf = [0u8; 4];
        iondb_core::endian::write_u32_le(&mut crc_buf, checksum)?;
        let checksum_offset = self.page_offset + data_len as u64;
        let _w = backend.write(checksum_offset, &crc_buf)?;
        Ok(())
    }

    /// Start a new page: write header.
    fn start_new_page(&mut self, backend: &mut impl IoBackend) -> Result<()> {
        if self.next_page_id > 0 {
            // Move to next page
            self.page_offset += self.page_size as u64;
        }
        let header = PageHeader::new(PageType::WalSegment, self.next_page_id);
        let mut hdr_buf = [0u8; PAGE_HEADER_SIZE];
        header.encode(&mut hdr_buf)?;
        let _w = backend.write(self.page_offset, &hdr_buf)?;
        self.pos_in_page = PAGE_HEADER_SIZE;
        self.next_page_id = self.next_page_id.wrapping_add(1);
        Ok(())
    }

    /// Current write offset in the backend.
    pub fn current_offset(&self) -> u64 {
        self.page_offset + self.pos_in_page as u64
    }

    /// Page offset of the current page.
    pub fn page_offset(&self) -> u64 {
        self.page_offset
    }

    /// Position within the current page.
    pub fn pos_in_page(&self) -> usize {
        self.pos_in_page
    }

    /// Next page_id.
    pub fn next_page_id(&self) -> PageId {
        self.next_page_id
    }
}

/// Read the next record from a page-segmented layout.
///
/// `page_offset` and `pos_in_page` describe the current read position.
/// Returns `Some((record, new_page_offset, new_pos_in_page))` or `None` at end.
///
/// If a page checksum fails, returns `Err(Error::Corruption)`. The caller
/// should advance to the next page: `page_offset += page_size, pos_in_page = PAGE_HEADER_SIZE`.
pub fn read_record_paged<'buf>(
    backend: &impl IoBackend,
    page_offset: u64,
    pos_in_page: usize,
    page_size: usize,
    end_offset: u64,
    buf: &'buf mut [u8],
) -> Result<Option<(record::WalRecord<'buf>, u64, usize)>> {
    if page_offset >= end_offset {
        return Ok(None);
    }

    let usable_end = page_size - PAGE_CHECKSUM_SIZE;

    // If we're past the usable area, move to next page
    if pos_in_page >= usable_end {
        let next_page = page_offset + page_size as u64;
        if next_page >= end_offset {
            return Ok(None);
        }
        return read_record_paged(backend, next_page, PAGE_HEADER_SIZE, page_size, end_offset, buf);
    }

    let abs_offset = page_offset + pos_in_page as u64;

    // Check if there's room for at least a header
    let remaining = usable_end - pos_in_page;
    if remaining < RECORD_HEADER_SIZE {
        // Padding — skip to next page
        let next_page = page_offset + page_size as u64;
        if next_page >= end_offset {
            return Ok(None);
        }
        return read_record_paged(backend, next_page, PAGE_HEADER_SIZE, page_size, end_offset, buf);
    }

    // Read record header
    let header_read = backend.read(abs_offset, &mut buf[..RECORD_HEADER_SIZE])?;
    if header_read < RECORD_HEADER_SIZE {
        return Ok(None);
    }

    // Check for magic — if not present, this is padding/end of records in page
    if buf[0] != record::MAGIC[0] || buf[1] != record::MAGIC[1] {
        // No more records in this page, try next
        let next_page = page_offset + page_size as u64;
        if next_page >= end_offset {
            return Ok(None);
        }
        return read_record_paged(backend, next_page, PAGE_HEADER_SIZE, page_size, end_offset, buf);
    }

    // Read key_len and val_len
    let key_len = iondb_core::endian::read_u16_le(&buf[23..])? as usize;
    let val_len = iondb_core::endian::read_u32_le(&buf[25..])? as usize;
    let total = record::record_size(key_len, val_len);

    // Check it fits in usable space
    if pos_in_page + total > usable_end {
        return Ok(None); // Truncated or corrupt — treat as end of page
    }

    if buf.len() < total {
        return Err(Error::WalError);
    }

    // Read remaining payload
    if total > RECORD_HEADER_SIZE {
        let payload_offset = abs_offset + RECORD_HEADER_SIZE as u64;
        let payload_read = backend.read(payload_offset, &mut buf[RECORD_HEADER_SIZE..total])?;
        if payload_read < total - RECORD_HEADER_SIZE {
            return Ok(None);
        }
    }

    // Deserialize with CRC validation
    let rec = record::deserialize_from(&buf[..total])?;
    let new_pos = pos_in_page + total;
    Ok(Some((rec, page_offset, new_pos)))
}

/// Verify the checksum of a page. Returns `Ok(())` if valid.
pub fn verify_page(backend: &impl IoBackend, page_offset: u64, page_size: usize) -> Result<()> {
    // Read page data (everything before checksum)
    let data_len = page_size - PAGE_CHECKSUM_SIZE;
    let mut crc_state = 0xFFFF_FFFFu32;
    let mut chunk = [0u8; 64];
    let mut read_off = page_offset;
    let end_off = page_offset + data_len as u64;
    while read_off < end_off {
        let to_read = chunk.len().min((end_off - read_off) as usize);
        let n = backend.read(read_off, &mut chunk[..to_read])?;
        if n == 0 {
            break;
        }
        crc_state = iondb_core::crc::crc32_update(crc_state, &chunk[..n]);
        read_off += n as u64;
    }
    let computed = crc_state ^ 0xFFFF_FFFF;

    // Read stored checksum
    let mut crc_buf = [0u8; 4];
    let _n = backend.read(page_offset + data_len as u64, &mut crc_buf)?;
    let stored = iondb_core::endian::read_u32_le(&crc_buf)?;

    if computed == stored {
        Ok(())
    } else {
        Err(Error::Corruption)
    }
}

//! Core WAL implementation: construction, append, sync, checkpoint, queries.

use iondb_core::error::{Error, Result};
use iondb_core::traits::io_backend::IoBackend;
use iondb_core::types::{Lsn, TxnId};

use crate::config::{SyncPolicy, TruncationMode, WalConfig, WalLayout};
use crate::flat;
use crate::paged::PagedWriter;
use crate::record::{self, RecordType};
use crate::recovery::{CommittedRecoveryReader, RawRecoveryReader};
#[cfg(feature = "alloc")]
use crate::recovery::OwnedWalRecord;

/// Internal layout state.
enum LayoutState {
    Flat {
        write_offset: u64,
    },
    Paged {
        writer: PagedWriter,
    },
}

/// The write-ahead log, generic over I/O backend.
pub struct Wal<I: IoBackend> {
    backend: I,
    config: WalConfig,
    layout: LayoutState,
    next_lsn: Lsn,
    checkpoint_lsn: Lsn,
    /// Record count since last sync (for Periodic policy).
    records_since_sync: u32,
}

impl<I: IoBackend> Wal<I> {
    /// Create a new, empty WAL.
    ///
    /// Validates the config and initializes the backend. For circular mode,
    /// writes the initial circular header.
    ///
    /// # Errors
    ///
    /// - [`Error::WalError`] if config is invalid.
    /// - [`Error::Io`] if the backend write fails.
    pub fn new(backend: I, config: WalConfig) -> Result<Self> {
        config.validate()?;
        let mut wal = Self {
            backend,
            config,
            layout: LayoutState::Flat { write_offset: 0 },
            next_lsn: 0,
            checkpoint_lsn: 0,
            records_since_sync: 0,
        };
        wal.init_layout()?;
        Ok(wal)
    }

    /// Open an existing WAL. Scans to find current write position and
    /// checkpoint LSN. Does NOT replay records.
    ///
    /// # Errors
    ///
    /// - [`Error::WalError`] if config is invalid or backend is empty.
    pub fn open(backend: I, config: WalConfig) -> Result<Self> {
        config.validate()?;
        let size = backend.size()?;
        if size == 0 {
            return Err(Error::WalError);
        }

        let mut wal = Self {
            backend,
            config,
            layout: LayoutState::Flat { write_offset: 0 },
            next_lsn: 0,
            checkpoint_lsn: 0,
            records_since_sync: 0,
        };
        wal.scan_existing()?;
        Ok(wal)
    }

    fn init_layout(&mut self) -> Result<()> {
        match (&self.config.layout, &self.config.truncation) {
            (WalLayout::Flat, TruncationMode::Circular { .. }) => {
                let header = flat::CircularHeader {
                    head_offset: flat::CIRCULAR_HEADER_SIZE as u64,
                    tail_offset: flat::CIRCULAR_HEADER_SIZE as u64,
                    checkpoint_lsn: 0,
                };
                flat::write_circular_header(&mut self.backend, &header)?;
                self.backend.sync()?;
                self.layout = LayoutState::Flat {
                    write_offset: flat::CIRCULAR_HEADER_SIZE as u64,
                };
            }
            (WalLayout::Flat, _) => {
                self.layout = LayoutState::Flat { write_offset: 0 };
            }
            (WalLayout::PageSegmented { page_size }, _) => {
                self.layout = LayoutState::Paged {
                    writer: PagedWriter::new(*page_size, 0),
                };
            }
        }
        Ok(())
    }

    fn scan_existing(&mut self) -> Result<()> {
        match &self.config.layout {
            WalLayout::Flat => self.scan_flat(),
            WalLayout::PageSegmented { page_size } => self.scan_paged(*page_size),
        }
    }

    fn scan_flat(&mut self) -> Result<()> {
        let start = match &self.config.truncation {
            TruncationMode::Circular { .. } => {
                let header = flat::read_circular_header(&self.backend)?;
                self.checkpoint_lsn = header.checkpoint_lsn;
                self.layout = LayoutState::Flat {
                    write_offset: header.head_offset,
                };
                // Scan from tail to find all records and their LSNs
                header.tail_offset
            }
            _ => {
                self.layout = LayoutState::Flat { write_offset: 0 };
                0
            }
        };

        let end = self.backend.size()?;
        let mut offset = start;
        let mut header_buf = [0u8; record::RECORD_HEADER_SIZE];
        let mut last_checkpoint_lsn: Option<Lsn> = None;
        let mut max_lsn: Lsn = 0;

        loop {
            if offset + record::RECORD_HEADER_SIZE as u64 > end {
                break;
            }
            let n = self.backend.read(offset, &mut header_buf)?;
            if n < record::RECORD_HEADER_SIZE {
                break;
            }
            match record::read_header(&header_buf) {
                Ok((lsn, _txn_id, rt, key_len, val_len)) => {
                    let total = record::record_size(key_len as usize, val_len as usize);
                    if offset + total as u64 > end {
                        break; // Truncated record
                    }
                    if lsn >= max_lsn {
                        max_lsn = lsn + 1; // next_lsn is one past the max
                    }
                    if rt == RecordType::Checkpoint {
                        last_checkpoint_lsn = Some(lsn);
                    }
                    offset += total as u64;
                }
                Err(_) => break, // End of valid records
            }
        }

        if !matches!(self.config.truncation, TruncationMode::Circular { .. }) {
            self.layout = LayoutState::Flat {
                write_offset: offset,
            };
        }
        self.next_lsn = max_lsn;
        if let Some(cp_lsn) = last_checkpoint_lsn {
            self.checkpoint_lsn = cp_lsn;
        }
        Ok(())
    }

    fn scan_paged(&mut self, page_size: usize) -> Result<()> {
        let end = self.backend.size()?;
        let mut page_offset = 0u64;
        let mut max_lsn: Lsn = 0;
        let mut last_checkpoint_lsn: Option<Lsn> = None;
        let mut last_page_offset = 0u64;
        let mut last_pos_in_page = iondb_core::page::PAGE_HEADER_SIZE;
        let mut last_page_id: u32 = 0;

        while page_offset + page_size as u64 <= end {
            // Try to read records from this page
            let mut pos = iondb_core::page::PAGE_HEADER_SIZE;
            let usable_end = page_size - iondb_core::page::PAGE_CHECKSUM_SIZE;

            loop {
                if pos + record::RECORD_HEADER_SIZE > usable_end {
                    break;
                }
                let abs = page_offset + pos as u64;
                let mut hdr = [0u8; record::RECORD_HEADER_SIZE];
                let n = self.backend.read(abs, &mut hdr)?;
                if n < record::RECORD_HEADER_SIZE {
                    break;
                }
                match record::read_header(&hdr) {
                    Ok((lsn, _txn_id, rt, key_len, val_len)) => {
                        let total = record::record_size(key_len as usize, val_len as usize);
                        if pos + total > usable_end {
                            break;
                        }
                        if lsn >= max_lsn {
                            max_lsn = lsn + 1;
                        }
                        if rt == RecordType::Checkpoint {
                            last_checkpoint_lsn = Some(lsn);
                        }
                        pos += total;
                        last_page_offset = page_offset;
                        last_pos_in_page = pos;
                    }
                    Err(_) => break,
                }
            }

            // Read page header to get page_id
            let mut phdr = [0u8; iondb_core::page::PAGE_HEADER_SIZE];
            let _n = self.backend.read(page_offset, &mut phdr)?;
            if let Ok(header) = iondb_core::page::PageHeader::decode(&phdr) {
                last_page_id = header.page_id.wrapping_add(1);
            }

            page_offset += page_size as u64;
        }

        self.layout = LayoutState::Paged {
            writer: PagedWriter::resume(page_size, last_page_offset, last_pos_in_page, last_page_id),
        };
        self.next_lsn = max_lsn;
        if let Some(cp_lsn) = last_checkpoint_lsn {
            self.checkpoint_lsn = cp_lsn;
        }
        Ok(())
    }

    // --- Append methods ---

    /// Write a Begin record for the given transaction.
    pub fn begin_tx(&mut self, txn_id: TxnId) -> Result<Lsn> {
        self.append_record(txn_id, RecordType::Begin, &[], &[])
    }

    /// Append a Put record.
    pub fn put(&mut self, txn_id: TxnId, key: &[u8], value: &[u8]) -> Result<Lsn> {
        self.append_record(txn_id, RecordType::Put, key, value)
    }

    /// Append a Delete record.
    pub fn delete(&mut self, txn_id: TxnId, key: &[u8]) -> Result<Lsn> {
        self.append_record(txn_id, RecordType::Delete, key, &[])
    }

    /// Write a Commit record.
    pub fn commit_tx(&mut self, txn_id: TxnId) -> Result<Lsn> {
        self.append_record(txn_id, RecordType::Commit, &[], &[])
    }

    /// Write a Rollback record.
    pub fn rollback_tx(&mut self, txn_id: TxnId) -> Result<Lsn> {
        self.append_record(txn_id, RecordType::Rollback, &[], &[])
    }

    fn append_record(
        &mut self,
        txn_id: TxnId,
        record_type: RecordType,
        key: &[u8],
        value: &[u8],
    ) -> Result<Lsn> {
        let lsn = self.next_lsn;
        let total = record::record_size(key.len(), value.len());

        // Serialize into a stack buffer. Max practical record is bounded
        // by MAX_KEY_LEN (256) + MAX_VALUE_LEN (65536) + 29 = 65821.
        // For no_std without alloc, we limit to a reasonable stack size.
        // In practice, most records are small. For large records the caller
        // should ensure the WAL has capacity.
        //
        // We use a fixed buffer here. If the record is too large, error out.
        // A more sophisticated approach could write in chunks, but YAGNI.
        const MAX_RECORD_BUF: usize = 512;
        if total > MAX_RECORD_BUF {
            return Err(Error::WalError);
        }
        let mut buf = [0u8; MAX_RECORD_BUF];
        let n = record::serialize_into(&mut buf, lsn, txn_id, record_type, key, value)?;

        match &mut self.layout {
            LayoutState::Flat { write_offset } => {
                // Check circular capacity
                if let TruncationMode::Circular { capacity } = self.config.truncation {
                    let head = *write_offset;
                    let tail = self.get_circular_tail()?;
                    let free = circular_free_space(head, tail, capacity);
                    if total > free {
                        return Err(Error::CapacityExhausted);
                    }
                    // Handle wrap-around
                    let header_size = flat::CIRCULAR_HEADER_SIZE as u64;
                    if head + total as u64 > capacity as u64 {
                        // Wrap to start of data area
                        *write_offset = header_size;
                    }
                }
                let new_offset = flat::write_record(&mut self.backend, *write_offset, &buf[..n])?;
                *write_offset = new_offset;
            }
            LayoutState::Paged { writer } => {
                let _off = writer.write_record(&mut self.backend, &buf[..n])?;
            }
        }

        self.next_lsn = lsn + 1;
        self.records_since_sync += 1;
        self.maybe_sync(record_type)?;
        Ok(lsn)
    }

    fn get_circular_tail(&self) -> Result<u64> {
        let header = flat::read_circular_header(&self.backend)?;
        Ok(header.tail_offset)
    }

    fn maybe_sync(&mut self, record_type: RecordType) -> Result<()> {
        let should_sync = match self.config.sync_policy {
            SyncPolicy::EveryRecord => true,
            SyncPolicy::EveryTransaction => matches!(
                record_type,
                RecordType::Commit | RecordType::Rollback
            ),
            SyncPolicy::Periodic(n) => self.records_since_sync >= n,
            SyncPolicy::None => false,
        };
        if should_sync {
            self.backend.sync()?;
            self.records_since_sync = 0;
        }
        Ok(())
    }

    // --- Sync and checkpoint ---

    /// Force sync to backend regardless of sync policy.
    pub fn sync(&mut self) -> Result<()> {
        self.backend.sync()?;
        self.records_since_sync = 0;
        Ok(())
    }

    /// Write a Checkpoint record and truncate per configured mode.
    ///
    /// The caller must ensure all dirty pages up to `up_to_lsn` have been
    /// flushed to storage before calling this.
    pub fn checkpoint(&mut self, up_to_lsn: Lsn) -> Result<()> {
        // Write checkpoint record (txn_id 0 for checkpoint)
        let _lsn = self.append_record(0, RecordType::Checkpoint, &[], &[])?;
        self.checkpoint_lsn = up_to_lsn;

        match self.config.truncation {
            TruncationMode::Logical => {
                // Nothing to do — checkpoint_lsn is tracked in memory
                // and discovered by scanning on open()
            }
            #[cfg(feature = "std")]
            TruncationMode::Physical => {
                self.physical_compact()?;
            }
            TruncationMode::Circular { .. } => {
                // Update circular header with new tail
                self.update_circular_checkpoint(up_to_lsn)?;
            }
        }
        Ok(())
    }

    /// Physical truncation: crash-safe two-phase compaction.
    ///
    /// 1. Write checkpoint pointer (LSN + data offset) at offset 0, sync.
    /// 2. Copy live data from checkpoint offset to beginning.
    /// 3. Update write_offset and checkpoint pointer. Sync.
    ///
    /// If crash occurs during step 2, recovery reads the checkpoint pointer
    /// and finds valid data at the original (pre-compaction) offsets.
    #[cfg(feature = "std")]
    fn physical_compact(&mut self) -> Result<()> {
        let write_end = match &self.layout {
            LayoutState::Flat { write_offset } => *write_offset,
            LayoutState::Paged { writer } => writer.current_offset(),
        };

        // Find the byte offset of the first record after checkpoint_lsn
        // by scanning from the start
        let mut scan_offset = 0u64;
        let mut checkpoint_byte_offset = 0u64;
        let mut header_buf = [0u8; record::RECORD_HEADER_SIZE];
        loop {
            if scan_offset + record::RECORD_HEADER_SIZE as u64 > write_end {
                break;
            }
            let n = self.backend.read(scan_offset, &mut header_buf)?;
            if n < record::RECORD_HEADER_SIZE {
                break;
            }
            match record::read_header(&header_buf) {
                Ok((lsn, _, _, key_len, val_len)) => {
                    let total = record::record_size(key_len as usize, val_len as usize);
                    if lsn > self.checkpoint_lsn {
                        checkpoint_byte_offset = scan_offset;
                        break;
                    }
                    scan_offset += total as u64;
                }
                Err(_) => break,
            }
        }

        if checkpoint_byte_offset == 0 {
            return Ok(()); // Nothing to compact
        }

        // Step 1: Write checkpoint pointer at offset 0 (reuse circular header format)
        // This is the commit point for crash safety
        let pointer = flat::CircularHeader {
            head_offset: write_end,
            tail_offset: checkpoint_byte_offset,
            checkpoint_lsn: self.checkpoint_lsn,
        };
        flat::write_circular_header(&mut self.backend, &pointer)?;
        self.backend.sync()?;

        // Step 2: Copy live data to beginning
        let live_len = write_end - checkpoint_byte_offset;
        let mut copy_buf = [0u8; 256];
        let mut src = checkpoint_byte_offset;
        let mut dst = 0u64;
        let mut remaining = live_len;
        while remaining > 0 {
            let chunk = copy_buf.len().min(remaining as usize);
            let n = self.backend.read(src, &mut copy_buf[..chunk])?;
            if n == 0 {
                break;
            }
            let _w = self.backend.write(dst, &copy_buf[..n])?;
            src += n as u64;
            dst += n as u64;
            remaining -= n as u64;
        }

        // Step 3: Update write offset
        match &mut self.layout {
            LayoutState::Flat { write_offset } => *write_offset = dst,
            _ => {}
        }
        self.backend.sync()?;
        Ok(())
    }

    fn update_circular_checkpoint(&mut self, _up_to_lsn: Lsn) -> Result<()> {
        // Find the offset corresponding to the checkpoint LSN by scanning
        // For now, update the header with the current checkpoint_lsn
        let write_offset = match &self.layout {
            LayoutState::Flat { write_offset } => *write_offset,
            _ => return Err(Error::WalError),
        };
        let header = flat::CircularHeader {
            head_offset: write_offset,
            tail_offset: flat::CIRCULAR_HEADER_SIZE as u64, // simplified for now
            checkpoint_lsn: self.checkpoint_lsn,
        };
        flat::write_circular_header(&mut self.backend, &header)?;
        self.backend.sync()?;
        Ok(())
    }

    /// Create a raw recovery reader over this WAL.
    pub fn recover(&self) -> Result<RawRecoveryReader<'_, I>> {
        Ok(RawRecoveryReader::new(
            &self.backend,

            self.recovery_start(),
            self.write_end(),
        ))
    }

    pub fn recover_committed<'a>(

    /// Collect all committed records into a Vec.
    #[cfg(feature = "alloc")]
    pub fn recover_committed_to_vec(&self) -> Result<alloc::vec::Vec<OwnedWalRecord>> {
        let mut scratch = alloc::vec![0u64; 256];
        let mut reader = self.recover_committed(&mut scratch)?;
        let mut buf = alloc::vec![0u8; 512];
        let mut records = alloc::vec::Vec::new();
        while let Some(rec) = reader.next_record(&mut buf)? {
            records.push(OwnedWalRecord::from_borrowed(&rec));
        }
        Ok(records)
    }

    /// Next LSN that will be assigned.
    pub fn current_lsn(&self) -> Lsn {

    /// Remaining capacity in bytes. `None` if unbounded.
    pub fn remaining(&self) -> Option<usize> {
        match self.config.truncation {
            TruncationMode::Circular { capacity } => {
                let head = match &self.layout {
                    LayoutState::Flat { write_offset } => *write_offset,
                    _ => return Some(0),
                };
                let tail = flat::read_circular_header(&self.backend)
                    .map(|h| h.tail_offset)
                    .unwrap_or(flat::CIRCULAR_HEADER_SIZE as u64);
                Some(circular_free_space(head, tail, capacity))
            }
            _ => None,
        }
    }

    /// Borrow the backend (for recovery readers).
    pub fn backend(&self) -> &I {
        &self.backend
    }

    /// Get the current write offset (for recovery end bound).
    pub fn write_end(&self) -> u64 {
        match &self.layout {
            LayoutState::Flat { write_offset } => *write_offset,
            LayoutState::Paged { writer } => writer.current_offset(),
        }
    }

    /// Get the layout config.
    pub fn layout(&self) -> &WalLayout {
        &self.config.layout
    }

    /// Get the recovery start offset (after checkpoint).
    pub fn recovery_start(&self) -> u64 {
        // For now, always start from 0. A proper implementation would
        // scan for the checkpoint record and start after it.
        // This is refined in the checkpoint/recovery tasks.
        match &self.config.truncation {
            TruncationMode::Circular { .. } => {
                flat::read_circular_header(&self.backend)
                    .map(|h| h.tail_offset)
                    .unwrap_or(flat::CIRCULAR_HEADER_SIZE as u64)
            }
            _ => 0,
        }
    }
}

/// Compute free space in a circular buffer.
fn circular_free_space(head: u64, tail: u64, capacity: usize) -> usize {
    let cap = capacity as u64;
    if head >= tail {
        (cap - head + tail - flat::CIRCULAR_HEADER_SIZE as u64) as usize
    } else {
        (tail - head) as usize
    }
}

//! Recovery readers: raw and committed-only, with zero-alloc `no_std` support.
//!
//! Recovery readers use `next_record()` instead of the `Iterator` trait.
//! This avoids the lending iterator problem: `WalRecord` borrows from the
//! caller-provided buffer, which is overwritten on each call.

use iondb_core::error::{Error, Result};
use iondb_core::page::PAGE_HEADER_SIZE;
use iondb_core::traits::io_backend::IoBackend;
use iondb_core::types::{Lsn, TxnId};

use crate::config::WalLayout;
use crate::flat;
use crate::paged;
use crate::record::{self, RecordType, WalRecord, RECORD_HEADER_SIZE};

/// Reads all valid records from checkpoint_lsn forward.
///
/// Skips corrupted records. For flat layout, uses magic-byte scanning
/// after corruption. For page-segmented, skips to the next page.
pub struct RawRecoveryReader<'a, I: IoBackend> {
    backend: &'a I,
    layout: &'a WalLayout,
    // Position tracking depends on layout
    offset: u64,
    end: u64,
    // For paged layout
    page_offset: u64,
    pos_in_page: usize,
    done: bool,
}

impl<'a, I: IoBackend> RawRecoveryReader<'a, I> {
    /// Create a new raw recovery reader.
    pub fn new(backend: &'a I, layout: &'a WalLayout, start: u64, end: u64) -> Self {
        let (page_offset, pos_in_page) = match layout {
            WalLayout::PageSegmented { .. } => (start, PAGE_HEADER_SIZE),
            WalLayout::Flat => (0, 0),
        };
        Self {
            backend,
            layout,
            offset: start,
            end,
            page_offset,
            pos_in_page,
            done: false,
        }
    }

    /// Read the next valid record into `buf`.
    ///
    /// Returns `Ok(None)` at end-of-log. Each call overwrites `buf`.
    /// The returned `WalRecord` borrows key/value slices from `buf`.
    pub fn next_record<'buf>(
        &mut self,
        buf: &'buf mut [u8],
    ) -> Result<Option<WalRecord<'buf>>> {
        if self.done {
            return Ok(None);
        }

        match self.layout {
            WalLayout::Flat => self.next_flat(buf),
            WalLayout::PageSegmented { .. } => self.next_paged(buf),
        }
    }

    fn next_flat<'buf>(&mut self, buf: &'buf mut [u8]) -> Result<Option<WalRecord<'buf>>> {
        loop {
            match flat::read_record(self.backend, self.offset, self.end, buf) {

                Ok(Some((rec, next_offset))) => {
                    self.offset = next_offset;
                    return Ok(Some(rec));
                }
                Ok(None) => {
                    self.done = true;
                    return Ok(None);
                }
                Err(Error::Corruption) => {
                    // Magic-byte scan to find next record
                    match flat::scan_for_magic(
                        self.backend,
                        self.offset + 1,
                        self.end,
                    )? {
                        Some(next) => self.offset = next,
                        None => {
                            self.done = true;
                            return Ok(None);
                        }
                    }
                }
                Err(e) => return Err(e),
            }
        }
    }

    fn next_paged<'buf>(
        &mut self,
        buf: &'buf mut [u8],
    ) -> Result<Option<WalRecord<'buf>>> {
        let page_size = match self.layout {
            WalLayout::PageSegmented { page_size } => *page_size,
            _ => return Ok(None),
        };
        loop {
            match paged::read_record_paged(
                self.backend,
                self.page_offset,
                self.pos_in_page,
                page_size,
                self.end,
                buf,
            ) {
                Ok(Some((rec, new_page_off, new_pos))) => {
                    self.page_offset = new_page_off;
                    self.pos_in_page = new_pos;
                    return Ok(Some(rec));
                }
                Ok(None) => {
                    self.done = true;
                    return Ok(None);
                }
                Err(Error::Corruption) => {
                    // Skip to next page
                    self.page_offset += page_size as u64;
                    self.pos_in_page = PAGE_HEADER_SIZE;
                    if self.page_offset >= self.end {
                        self.done = true;
                        return Ok(None);
                    }
                }
                Err(e) => return Err(e),
            }
        }
    }
}

/// Reads only records from committed transactions.
///
/// Two-pass approach:
/// - Pass 1 (at construction): scans record headers to find committed TxnIds.
/// - Pass 2 (next_record calls): yields records whose TxnId is committed.
pub struct CommittedRecoveryReader<'a, I: IoBackend> {
    backend: &'a I,
    layout: &'a WalLayout,
    /// Committed transaction IDs (filled during pass 1).
    scratch: &'a [TxnId],
    /// Number of committed TxnIds found.
    committed_count: usize,
    // Position tracking for pass 2
    offset: u64,
    end: u64,
    page_offset: u64,
    pos_in_page: usize,
    done: bool,
}

impl<'a, I: IoBackend> CommittedRecoveryReader<'a, I> {
    /// Create a new committed recovery reader.
    ///
    /// Pass 1 runs immediately: scans the log to find committed TxnIds.
    ///
    /// # Errors
    ///
    /// - [`Error::WalError`] if more in-flight transactions exist than
    ///   `scratch` can hold.
    pub fn new(
        backend: &'a I,
        layout: &'a WalLayout,
        start: u64,
        end: u64,
        scratch: &'a mut [TxnId],
    ) -> Result<Self> {
        // Pass 1: scan headers to find committed TxnIds
        let committed_count = Self::scan_committed(backend, layout, start, end, scratch)?;

        let (page_offset, pos_in_page) = match layout {
            WalLayout::PageSegmented { .. } => (start, PAGE_HEADER_SIZE),
            WalLayout::Flat => (0, 0),
        };

        Ok(Self {
            backend,
            layout,
            scratch,
            committed_count,
            offset: start,
            end,
            page_offset,
            pos_in_page,
            done: false,
        })
    }

    fn scan_committed(
        backend: &I,
        layout: &WalLayout,
        start: u64,
        end: u64,
        scratch: &mut [TxnId],
    ) -> Result<usize> {
        let mut count = 0usize;
        let mut header_buf = [0u8; RECORD_HEADER_SIZE];

        match layout {
            WalLayout::Flat => {
                let mut offset = start;
                loop {
                    if offset + RECORD_HEADER_SIZE as u64 > end {
                        break;
                    }
                    let n = backend.read(offset, &mut header_buf)?;
                    if n < RECORD_HEADER_SIZE {
                        break;
                    }
                    match record::read_header(&header_buf) {
                        Ok((_, txn_id, rt, key_len, val_len)) => {
                            let total = record::record_size(key_len as usize, val_len as usize);
                            if rt == RecordType::Commit {
                                if count >= scratch.len() {
                                    return Err(Error::WalError);
                                }
                                scratch[count] = txn_id;
                                count += 1;
                            }
                            offset += total as u64;
                        }
                        Err(_) => break,
                    }
                }
            }
            WalLayout::PageSegmented { page_size } => {
                let page_size = *page_size;
                let mut page_off = start;
                while page_off + page_size as u64 <= end {
                    let mut pos = PAGE_HEADER_SIZE;
                    let usable_end = page_size - iondb_core::page::PAGE_CHECKSUM_SIZE;
                    loop {
                        if pos + RECORD_HEADER_SIZE > usable_end {
                            break;
                        }
                        let abs = page_off + pos as u64;
                        let n = backend.read(abs, &mut header_buf)?;
                        if n < RECORD_HEADER_SIZE {
                            break;
                        }
                        match record::read_header(&header_buf) {
                            Ok((_, txn_id, rt, key_len, val_len)) => {
                                let total =
                                    record::record_size(key_len as usize, val_len as usize);
                                if rt == RecordType::Commit {
                                    if count >= scratch.len() {
                                        return Err(Error::WalError);
                                    }
                                    scratch[count] = txn_id;
                                    count += 1;
                                }
                                pos += total;
                            }
                            Err(_) => break,
                        }
                    }
                    page_off += page_size as u64;
                }
            }
        }

        Ok(count)
    }

    fn is_committed(&self, txn_id: TxnId) -> bool {
        self.scratch[..self.committed_count].contains(&txn_id)
    }

    /// Read the next committed record into `buf`.
    ///
    /// Skips records from uncommitted/rolled-back transactions.
    pub fn next_record<'buf>(
        &mut self,
        buf: &'buf mut [u8],
    ) -> Result<Option<WalRecord<'buf>>> {
        if self.done {
            return Ok(None);
        }

        // Create a temporary raw reader state inline
        loop {
            let result = match self.layout {
                WalLayout::Flat => {
                    flat::read_record(self.backend, self.offset, self.end, buf)
                }
                WalLayout::PageSegmented { page_size } => {
                    match paged::read_record_paged(
                        self.backend,
                        self.page_offset,
                        self.pos_in_page,
                        page_size,
                        self.end,
                        buf,
                    ) {
                        Ok(Some((rec, new_page_off, new_pos))) => {
                            self.page_offset = new_page_off;
                            self.pos_in_page = new_pos;
                            Ok(Some((rec, 0))) // offset unused for paged
                        }
                        Ok(None) => Ok(None),
                        Err(e) => Err(e),
                    }
                }
            };

            match result {
                Ok(Some((rec, next_offset))) => {
                    if matches!(self.layout, WalLayout::Flat) {
                        self.offset = next_offset;
                    }
                    if self.is_committed(rec.txn_id) {
                        return Ok(Some(rec));
                    }
                    // Skip non-committed records, continue loop
                }
                Ok(None) => {
                    self.done = true;
                    return Ok(None);
                }
                Err(Error::Corruption) => {
                    match self.layout {
                        WalLayout::Flat => {
                            match flat::scan_for_magic(
                                self.backend,
                                self.offset + 1,
                                self.end,
                            )? {
                                Some(next) => self.offset = next,
                                None => {
                                    self.done = true;
                                    return Ok(None);
                                }
                            }
                        }
                        WalLayout::PageSegmented { page_size } => {
                            self.page_offset += page_size as u64;
                            self.pos_in_page = PAGE_HEADER_SIZE;
                            if self.page_offset >= self.end {
                                self.done = true;
                                return Ok(None);
                            }
                        }
                    }
                }
                Err(e) => return Err(e),
            }
        }
    }
}

use iondb_wal::record::RecordType;
use iondb_wal::recovery::{CommittedRecoveryReader, RawRecoveryReader};
use iondb_wal::wal::Wal;
use proptest::prelude::*;

/// Operations that can be performed on the WAL.
#[derive(Debug, Clone)]
enum Op {
    Begin(u64),
    Put(u64, Vec<u8>, Vec<u8>),
    Delete(u64, Vec<u8>),
    Commit(u64),
    Rollback(u64),
}

/// Strategy for generating WAL operations.
fn op_strategy() -> impl Strategy<Value = Vec<Op>> {
    // Generate a sequence of complete transactions
    let txn_strategy = (1..=5u64).prop_flat_map(|num_txns| {
        let mut ops = Vec::new();
        prop::collection::vec(
            prop::collection::vec(
                prop_oneof![
                    (prop::collection::vec(any::<u8>(), 1..8),
                     prop::collection::vec(any::<u8>(), 1..16))
                        .prop_map(|(k, v)| (true, k, v)),
                    prop::collection::vec(any::<u8>(), 1..8)
                        .prop_map(|k| (false, k, vec![])),
                ],
                1..5,
            ),
            1..num_txns as usize + 1,
        )
        .prop_flat_map(move |txn_ops| {
            // For each transaction, decide commit or rollback
            prop::collection::vec(prop::bool::ANY, txn_ops.len()).prop_map(
                move |commit_flags| {
                    let mut all_ops = Vec::new();
                    for (txn_idx, (mutations, should_commit)) in
                        txn_ops.iter().zip(commit_flags.iter()).enumerate()
                    {
                        let txn_id = txn_idx as u64 + 1;
                        all_ops.push(Op::Begin(txn_id));
                        for (is_put, key, value) in mutations {
                            if *is_put {
                                all_ops.push(Op::Put(txn_id, key.clone(), value.clone()));
                            } else {
                                all_ops.push(Op::Delete(txn_id, key.clone()));
                            }
                        }
                        if *should_commit {
                            all_ops.push(Op::Commit(txn_id));
                        } else {
                            all_ops.push(Op::Rollback(txn_id));
                        }
                    }
                    all_ops
                },
            )
        })
    });
    txn_strategy
}

fn apply_ops(wal: &mut Wal<impl iondb_core::IoBackend>, ops: &[Op]) {
    for op in ops {
        match op {
            Op::Begin(txn) => { let _ = wal.begin_tx(*txn); }
            Op::Put(txn, k, v) => { let _ = wal.put(*txn, k, v); }
            Op::Delete(txn, k) => { let _ = wal.delete(*txn, k); }
            Op::Commit(txn) => { let _ = wal.commit_tx(*txn); }
            Op::Rollback(txn) => { let _ = wal.rollback_tx(*txn); }
        }
    }
}

fn collect_committed_records(
    wal: &Wal<impl iondb_core::IoBackend>,
) -> Vec<(u64, u64, RecordType)> {
    let mut scratch = [0u64; 64];
    let mut reader = wal.recover_committed(&mut scratch).expect("reader");

    let mut buf = [0u8; 512];
    let mut records = Vec::new();
    while let Some(rec) = reader.next_record(&mut buf).expect("read") {
        records.push((rec.lsn, rec.txn_id, rec.record_type));
    }
    records
}

// ===== Layout Equivalence =====

proptest! {
    #![proptest_config(ProptestConfig::with_cases(32))]

    #[test]
    fn layout_equivalence(ops in op_strategy()) {
        // Run same ops on flat and paged, verify same committed records
        let mut storage_flat = vec![0u8; 65536];
        let backend_flat = MemoryIoBackend::new(&mut storage_flat);
        let mut wal_flat = Wal::new(backend_flat, WalConfig {
            layout: WalLayout::Flat,
            sync_policy: SyncPolicy::None,
            truncation: TruncationMode::Logical,
        }).expect("new flat");

        let mut storage_paged = vec![0u8; 65536];
        let backend_paged = MemoryIoBackend::new(&mut storage_paged);
        let mut wal_paged = Wal::new(backend_paged, WalConfig {
            layout: WalLayout::PageSegmented { page_size: 256 },
            sync_policy: SyncPolicy::None,
            truncation: TruncationMode::Logical,
        }).expect("new paged");

        apply_ops(&mut wal_flat, &ops);
        apply_ops(&mut wal_paged, &ops);

        let flat_records = collect_committed_records(&wal_flat);
        let paged_records = collect_committed_records(&wal_paged);

        // Same committed records (LSN values may differ due to internal records,
        // so compare txn_id + record_type sequences)
        let flat_txn: Vec<_> = flat_records.iter().map(|(_, t, r)| (*t, *r)).collect();
        let paged_txn: Vec<_> = paged_records.iter().map(|(_, t, r)| (*t, *r)).collect();
        prop_assert_eq!(flat_txn, paged_txn);
    }
}

proptest! {
    #![proptest_config(ProptestConfig::with_cases(64))]

    #[test]
    fn crash_recovery_flat(
        ops in op_strategy(),
        crash_after in 1..20u64,
    ) {
        let mut storage = vec![0u8; 65536];

        // Phase 1: write ops with fault injection
        {
            let backend = MemoryIoBackend::new(&mut storage);
            let mut failpoint = FailpointIoBackend::new(backend);
            failpoint.set_fault_after(Fault::ErrorBeforeWrite, crash_after);

            let config = WalConfig {
                layout: WalLayout::Flat,
                sync_policy: SyncPolicy::EveryTransaction,
                truncation: TruncationMode::Logical,
            };
            if let Ok(mut wal) = Wal::new(failpoint, config) {
                apply_ops(&mut wal, &ops);
            }
        }

        // Phase 2: recover
        let backend = MemoryIoBackend::with_len(&mut storage, storage.len() as u64)
            .expect("with_len");
        let config = WalConfig {
            layout: WalLayout::Flat,
            sync_policy: SyncPolicy::EveryTransaction,
            truncation: TruncationMode::Logical,
        };
        if let Ok(wal) = Wal::open(backend, config) {
            let records = collect_committed_records(&wal);

            // Verify invariants
            let mut prev_lsn = None;
            for (lsn, _txn_id, _rt) in &records {
                if let Some(prev) = prev_lsn {
                    prop_assert!(*lsn > prev, "LSNs must be monotonically increasing");
                }
                prev_lsn = Some(*lsn);
            }

            // All returned records should be from committed transactions
            // (CommittedRecoveryReader guarantees this)
        }
    }
}

// ===== Sync Policy x Crash Tests =====

#[test]
fn crash_with_every_transaction_sync_preserves_committed() {
    let mut storage = vec![0u8; 65536];

    // Write: begin, put, commit (synced), begin, put (crash before commit)
    {
        let backend = MemoryIoBackend::new(&mut storage);
        let mut failpoint = FailpointIoBackend::new(backend);
        // Crash on the 6th write (the second put)
        failpoint.set_fault_after(Fault::ErrorBeforeWrite, 5);

        let config = WalConfig {
            layout: WalLayout::Flat,
            sync_policy: SyncPolicy::EveryTransaction,
            truncation: TruncationMode::Logical,
        };
        if let Ok(mut wal) = Wal::new(failpoint, config) {
            let _ = wal.begin_tx(1);
            let _ = wal.put(1, b"k1", b"v1");
            let _ = wal.commit_tx(1); // synced
            let _ = wal.begin_tx(2);
            let _ = wal.put(2, b"k2", b"v2"); // may fail
        }
    }

    // Recover
    let backend = MemoryIoBackend::with_len(&mut storage, storage.len() as u64)
        .expect("with_len");
    let config = WalConfig {
        layout: WalLayout::Flat,
        sync_policy: SyncPolicy::EveryTransaction,
        truncation: TruncationMode::Logical,
    };
    if let Ok(wal) = Wal::open(backend, config) {
        let records = collect_committed_records(&wal);
        // Txn 1 should be recovered, txn 2 should not
        assert!(records.iter().all(|(_, txn_id, _)| *txn_id == 1));
        assert!(!records.is_empty());
    }
}

#[test]
fn crash_with_no_sync_may_lose_data() {
    let mut storage = vec![0u8; 65536];

    {
        let backend = MemoryIoBackend::new(&mut storage);
        // MemoryIoBackend sync is a no-op, so "no sync" behaves the same.
        // This test verifies the API works with SyncPolicy::None.
        let config = WalConfig {
            layout: WalLayout::Flat,
            sync_policy: SyncPolicy::None,
            truncation: TruncationMode::Logical,
        };
        let mut wal = Wal::new(backend, config).expect("new");
        let _ = wal.begin_tx(1);
        let _ = wal.put(1, b"k", b"v");
        let _ = wal.commit_tx(1);
        // No explicit sync — data is in memory backend (which persists)
    }

    let backend = MemoryIoBackend::with_len(&mut storage, storage.len() as u64)
        .expect("with_len");
    let config = WalConfig {
        layout: WalLayout::Flat,
        sync_policy: SyncPolicy::None,
        truncation: TruncationMode::Logical,
    };
    let wal = Wal::open(backend, config).expect("open");
    let records = collect_committed_records(&wal);
    // With memory backend, data persists even without sync
    assert!(!records.is_empty());
}

#![no_main]

use libfuzzer_sys::fuzz_target;
use iondb_io::memory::MemoryIoBackend;
use iondb_wal::config::{SyncPolicy, TruncationMode, WalConfig, WalLayout};
use iondb_wal::recovery::RawRecoveryReader;
use iondb_wal::wal::Wal;

fuzz_target!(|data: &[u8]| {
    if data.len() < 256 {
        return;
    }

    // Write some valid records, then corrupt with fuzzer data
    let mut storage = vec![0u8; 65536];
    let backend = MemoryIoBackend::new(&mut storage);
    let config = WalConfig {
        layout: WalLayout::Flat,
        sync_policy: SyncPolicy::None,
        truncation: TruncationMode::Logical,
    };

    if let Ok(mut wal) = Wal::new(backend, config.clone()) {
        let _ = wal.begin_tx(1);
        let _ = wal.put(1, b"key", b"value");
        let _ = wal.commit_tx(1);

        let end = wal.write_end();

        // Inject corruption from fuzzer
        let corrupt_offset = (data[0] as u64) % end.max(1);
        let corrupt_len = data.len().min((end - corrupt_offset) as usize).min(128);
        if corrupt_len > 0 {
            drop(wal);
            storage[corrupt_offset as usize..corrupt_offset as usize + corrupt_len]
                .copy_from_slice(&data[1..1 + corrupt_len]);

            // Try recovery — should not panic
            let backend = MemoryIoBackend::with_len(&mut storage, storage.len() as u64)
                .unwrap_or_else(|_| return);
            let mut reader = RawRecoveryReader::new(&backend, WalLayout::Flat, 0, end);
            let mut buf = [0u8; 512];
            while let Ok(Some(_)) = reader.next_record(&mut buf) {}
        }
    }
});

#![no_main]

use arbitrary::Arbitrary;
use libfuzzer_sys::fuzz_target;
use iondb_io::memory::MemoryIoBackend;
use iondb_wal::config::{SyncPolicy, TruncationMode, WalConfig, WalLayout};
use iondb_wal::recovery::CommittedRecoveryReader;
use iondb_wal::wal::Wal;

#[derive(Arbitrary, Debug)]
enum FuzzOp {
    Begin(u8),
    Put(u8, Vec<u8>, Vec<u8>),
    Delete(u8, Vec<u8>),
    Commit(u8),
    Rollback(u8),
    Checkpoint,
    Recover,
}

fuzz_target!(|ops: Vec<FuzzOp>| {
    let mut storage = vec![0u8; 65536];
    let backend = MemoryIoBackend::new(&mut storage);
    let config = WalConfig {
        layout: WalLayout::Flat,
        sync_policy: SyncPolicy::EveryTransaction,
        truncation: TruncationMode::Logical,
    };

    if let Ok(mut wal) = Wal::new(backend, config) {
        for op in &ops {
            match op {
                FuzzOp::Begin(t) => { let _ = wal.begin_tx(*t as u64); }
                FuzzOp::Put(t, k, v) => {
                    let k = &k[..k.len().min(32)];
                    let v = &v[..v.len().min(64)];
                    let _ = wal.put(*t as u64, k, v);
                }
                FuzzOp::Delete(t, k) => {
                    let k = &k[..k.len().min(32)];
                    let _ = wal.delete(*t as u64, k);
                }
                FuzzOp::Commit(t) => { let _ = wal.commit_tx(*t as u64); }
                FuzzOp::Rollback(t) => { let _ = wal.rollback_tx(*t as u64); }
                FuzzOp::Checkpoint => {
                    let lsn = wal.current_lsn().saturating_sub(1);
                    let _ = wal.checkpoint(lsn);
                }
                FuzzOp::Recover => {
                    let mut scratch = [0u64; 256];
                    if let Ok(mut reader) = CommittedRecoveryReader::new(
                        wal.backend(),
                        *wal.layout(),
                        wal.recovery_start(),
                        wal.write_end(),
                        &mut scratch,
                    ) {
                        let mut buf = [0u8; 512];
                        while let Ok(Some(_)) = reader.next_record(&mut buf) {}
                    }
                }
            }
        }
    }
});
```
